FAQ Page

What is image text recognition?

It is the process that converts an image of text into string that user can handle normally (cut, copy, edit,...).

What types of images that your website currently supporting?

My website currently accepts jpg, jpeg, png. Other image file formats may cause an error, including pdf. If you want to get text from a pdf file, please check out this

Are there any limitations to your image text recognition software?

Yes. You can upload up to 3 images for converting. If you want to convert more, please use another ip address to do that

Is your image text recognition software accurate?

It's quite accurate in English. However, there are some difficulties in recognizing Vietnamese characters. I'm trying my best to improve that.

Can your software recognize handwriting?

Definately! Since this program using the best models to even recognize inregular text pattern in scene, it should be no problem in recognizing handwriting.

Will my images and recognized text be kept private?

Yes! There is no access to any image that another user uploading in anyway.
